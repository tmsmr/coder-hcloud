## Coder

### TL;DR
- Copy `coder/config.auto.tfvars.example` to `coder/config.auto.tfvars`
- Adjust `coder/config.auto.tfvars`
- `cd coder && terraform init && terraform apply`
- Create DNS records
- Open Coder, login with initial admin user
